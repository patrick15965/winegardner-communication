## The core problem
Estimating and Operations run like "two different companies." Estimators bid with no visibility into
the ops schedule; bid→handoff meetings are messy and adversarial; concerns surface *after* a bid is
committed; standards (lead times, real production rates, regional wage rules) live in people's heads;
tasks are scattered across Monday.com / Excel / email with no accountability.
